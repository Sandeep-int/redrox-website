# REDROX Industries - Official Website

![REDROX - The Backbone of Better Buildings](https://redroxindustries.com/favicon.ico) > **Live Website:** [redroxindustries.com](https://redroxindustries.com)

This repository contains the source code for the official website of **REDROX Industries Pvt Ltd.** ## 🧱 About REDROX
REDROX is an innovative, clean-tech construction startup solving the global textile waste crisis. We manufacture sustainable, high-strength interlocking infrastructure blocks using shredded waste cloth fibers and 110-ton hydraulic compression technology. 

Our blocks deliver:
* **Strength:** 4-5x stronger than conventional bricks.
* **Speed:** Reduces construction timelines from 2 years to ~6 months via dry-stack interlocking.
* **Comfort:** Proven 10°C thermal insulation and advanced acoustic soundproofing.
* **Sustainability:** 70% reduction in water and cement usage per project.

## 💻 Tech Stack
This is a lightweight, high-performance static website built for speed and global accessibility.
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Form Handling:** [Formspree](https://formspree.io/) (Serverless form submissions)
* **Hosting & CI/CD:** [Netlify](https://www.netlify.com/) (Automated deployments from the `main` branch)
